1. jdk  jre  jvm的含义和关系
JDK java开发者工具包 是整个java开发的环境
jdk 包含  开发工具bin文件夹中的 javac.exe java.exe..     jre 运行时环境
jre 包含  jvm Java虚拟机  和 系统的核心类库
2. java技术的构成 各个技术适应的项目类型分别是什么
javaSE    核心技术  主要用于开发桌面应用程序  控制台程序 窗体程序
javaEE   企业级应用开发技术 主要用于开发 站点类型项目
javaME  微型项目开发技术   嵌入式开发 移动端开发
3. java数据类型的分类 和 各自包含的数据类型
基本数据类型  byte  short  int  long    float  double  char  boolean 
引用数据类型  除上面8种基本数据类型以外的其他类型 都是引用类型 String  数组 系统定义的类型 自定义的类
4. 循环的分类  各种循环的特点
while             先判断 后执行      执行0-N次     适合处理次数不确定的重复操作
do-while      先执行 后判断      执行 1-N次    适合处理次数不确定的重复操作
for                 先判断  后执行      执行0-N次  适合处理次数明确的重复操作
5. break关键字和continue关键字的区别
continue    只能在循环内使用                          跳出本次循环 继续循环下一次
break         可以在循环 或 switch语句块中使用   跳出或结束 整个当前语句块 执行语句块后的代码段
6. 变量命名的规则和规范有哪些
规则  变量名只能使用 字母 数字 $  _ 组成 其中数字不能作为首字符
         变量名不能与系统的关键字 保留字冲突  (goto)
         在同一个作用域中  变量不能重复声明
规范  小驼峰命名法  stuName warePrice
         见名知意  
         易用性  简写 缩写
标识符  开发者自己定义的 类名 方法名 包名 变量名

7. import和package的区别
package 用于声明包  描述当前类文件所属的包路径  必须是类中第一行非注释性代码 一个类只能有一次声明包
import 导包  用于描述当前类文件引入的其他类文件路径  一个类文件中可以导入多个包
