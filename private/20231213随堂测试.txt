1. 重载和重写的概念
重载:  在一个类 或形成继承关系的父子类中  存在两个或多个  方法名相同 参数列表不同的方法  形成方法重载
重写:  在形成继承关系的父子类中  子类拥有和父类 同名 同参数列表 同返回值类型  且 权限修饰级别不低于父类同名方法  此时子类方法
的主体 对父类方法的主体进行覆盖 称为重写
2. 面向对象有哪些特征  这些特征的含义是什么
封装  隐藏具体实现 通过方法名提供给用户进行调用
继承  一个类使用extends关键字关联另一个类  从而获取被关联类所有属性和除构造方法外所有方法的过程 称为继承
多态   一个对象的多种形态   一个父类类型的对象 可以存储不同子类实例 不同子类的实例中对父类的方法进行不同的重写实现 
3. 类继承具有哪些特性
单根性  一个子类只能继承自一个父类
传递性  父类的成员 子类 及 子类的子类都可以继承得到 
侵入性  子类在继承得到父类成员的过程中 不具备对继承成员的选择性 
4. 编写一个单例模式的类
public class Single{
	//构造方法私有化
	private Single(){}
	private static Single s=new Single();
	//只读
	public static Single getSingle(){
		return s;
	}
}
public class Single{
	//构造方法私有化
	private Single(){}
	private static Single s;
	//只读
	public static Single getSingle(){
		if(s==null){
			s=new Single();
		}
		return s;
	}
}
5. 静态成员和实例成员的区别
静态成员依附类    类加载就生成                  类名.成员变量  类名.成员方法(  )        具有唯一性             静态成员不能访问实例成员
实例成员依附对象 实例化生成对象才生成   实例化对象后  对象名.成员名         每个对象独享一份      实例成员可以访问静态成员

7. 类与对象的概念 关系
现实   万物皆对象   类是一组具有相同特征 和 相似行为 的对象集合    先有对象 抽离出类
程序   类是模子  对象是类的实例化结果    先定义类  后 实例化生成对象
8. this和super关键字的区别
this指代本类对象 和本类的成员引用    super指代本类的父类及父类成员引用
